服などの3Dモデルに座標を指定して好きな形状の穴を開けることが出来るシェーダーです。
lilToonをカスタムしたシェーダーなので、シェーダーがlilToonであればそのままご利用できます！
しっぽ穴や帽子の耳の貫通防止、またはその他の用途などに使えます。


概要
1マテリアルにつき4つまで穴を開けられます。
対応している形状は
円（楕円）、四角形、ダイヤモンド、ハート、アステロイド、星
の５種類を用意しています。
アステロイドと星には個別の調整パラメータがあります。

簡易的な縁取り機能が付属しています。


使い方
1. unitypackageをインポート
2. 穴を開けたいモデルのマテリアルを選択し、インスペクター上のシェーダー一覧→lilCustomClipper→lilToon を選択
3. 描画モードをカットアウトまたは半透明にする
4. カスタムプロパティから穴を開ける位置や形状、大きさを指定する

インスペクター下のマテリアルプレビューを見ながら操作すると分かりやすいと思います。


注意事項
・3DモデルのUVによっては正しく動作しないことがあります（特に服の背面でUVが途切れているパターン）。この場合は穴を合体させるなどして対応してください。

・カスタムシェーダーに変更後マテリアルがのっぺりすることがありますが、描画モードを変更すると直ります。
